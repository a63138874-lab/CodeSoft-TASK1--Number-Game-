# CodeSoft-TASK1--Number-Game-
A simple and interactive console-based Number Guessing Game built using Java. The game generates a random number within a given range, and the player must guess it within a limited number of attempts.

 ## Features
1.  Random number generation (1 to 100)
2. User input-based guessing
3. Feedback system:
       Too High
       Too Low
4. Correct Guess
      Limited number of attempts
      Multiple rounds support Score tracking system

🛠️ Technologies Used
Java
OOP Concepts
Java Util Package (Random, Scanner)

📂 Project Structure
NumberGame.java
README.md


🎯 How to Play
The system generates a random number between 1 and 100.
You need to guess the number.
After each guess, you’ll receive a hint:
📈 Too High
📉 Too Low
You have a limited number of attempts.
Score is calculated based on how quickly you guess the number.
You can choose to play multiple rounds.
📊 Scoring System
Fewer attempts = Higher score 
Score accumulates across multiple rounds
